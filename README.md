# baryonlabs/homebrew-talkmode

Homebrew Cask tap for [TalkMode](https://github.com/baryonlabs/talkmode) — Korean-friendly real-time voice assistant for macOS.

## Install

```bash
brew tap baryonlabs/talkmode
brew install --cask talkmode
```

## Upgrade

```bash
brew upgrade --cask talkmode
```

(TalkMode also self-updates through Sparkle once installed.)

## Uninstall

```bash
brew uninstall --cask --zap talkmode
```

`--zap` removes user data (`~/.talk_mode_mac/`) as well.

## Requirements

- macOS Sonoma (14.0) or later
- Apple silicon (arm64)
